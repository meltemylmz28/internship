# Frontend MVP

Bu klasör, React + TypeScript + Vite tabanlı bir SPA kurulumunu içerir.

## Kurulum

1. Node.js ve npm/yarn kurulmalı.
2. Aşağıdaki komutlar çalıştırılmalı:

```bash
npm install
npm run dev
```

## Özellikler
- Plan sihirbazı arayüzü
- Ders/sınıf/eğitim yılı giriş alanları
- Haftalık plan tablosu
